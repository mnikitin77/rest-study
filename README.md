# RESTful API sample project in Kotlin

### Frameworks and tools used include:
- Spring Boot
- Spring MVC
- OpenAPI 3 specification
- openapi-generator-maven-plugin by 
- springdoc-openapi
- Spring Data

### Some features:
- Polymorphic dto model generated by openapi-generator-maven-plugin by OpenAPI 3 spec
- Entity model inheritance
- Logging servlet filter reading request and response bodies

### Important notice
- `*.mustache` classes fix Kotlin code generation flaws
- `ContentCachingResponseWrapper.copyBodyToResponse()` is necessary to restore the response body
